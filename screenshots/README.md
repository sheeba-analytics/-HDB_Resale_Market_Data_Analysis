# Screenshots

This folder contains PNG images of the Power BI dashboards created for the HDB Resale Price Analysis project. 
These screenshots provide a quick visual overview of the analysis and insights.

## Files

**hdb_sales_overview.png** – Overview of HDB resale sales metrics and key trends.

**Trends_overtime.png** – Shows HDB resale price trends over time.

**Town_clusters.png** – Town clustering analysis based on pricing and other characteristics.

**town_comparison.png** – Comparison of towns across metrics like average price and transaction volume.

**Flat_Price_Analysis.png** – Impact of flat age and flat type on pricing.

**Town_region_analysis.png** – Visualization of pricing and transaction patterns across towns and regions.

## Note
These screenshots represent the visuals available in the interactive Power BI dashboard located in the **PowerBI** folder.
